<h2>Usage</h2>
Upon opening the application, you'll see the following features:
<ul> <li><strong>Toggle Light/Dark Mode</strong>: Click the light bulb icon to switch between light and dark themes.</li> <li><strong>Task Management</strong>: <ul> <li><strong>Create Task</strong>: Click the "+ Add Task" button to open a form and fill in the details.</li> <li><strong>Mark as Completed</strong>: Click on the task to mark it as completed, which will update its status.</li> <li><strong>Delete Task</strong>: Click the delete button next to any task to remove it.</li> </ul> </li> </ul>

<h2>Additional Features</h2>

<ul> <li><strong>Light/Dark Mode Toggle</strong>: Users can switch between light and dark modes for a personalized experience.</li> <li><strong>Task Management</strong>: Create, delete, and mark tasks as completed with intuitive buttons and notifications.</li> <li><strong>Responsive Design</strong>: The application is optimized for various screen sizes, ensuring usability on mobile and desktop devices.</li> <li><strong>Custom Hooks</strong>: Utilizes <code>useTaskService</code> for efficient task-related API interactions, enhancing code reusability.</li> <li><strong>User-Friendly Alerts</strong>: Integrated toast notifications provide feedback on user actions (e.g., task creation, deletion).</li> </ul>
<h2>Technologies Used</h2>
<ul> <li><strong>React</strong>: A JavaScript library for building user interfaces.</li> <li><strong>TypeScript</strong>: A typed superset of JavaScript that compiles to plain JavaScript.</li> <li><strong>Context</strong>: A predictable state container for JavaScript apps.</li> <li><strong>Axios</strong>: A promise-based HTTP client for making API requests.</li> <li><strong>React Icons</strong>: A library for including icons in the application.</li> <li><strong>SCSS</strong>: A preprocessor for CSS that allows for nested rules and variables.</li> <li><strong>Ant Design (Antd)</strong>: A React component library to create user-friendly UI.</li> </ul> ```
