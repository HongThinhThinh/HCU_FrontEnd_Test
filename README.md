Usage
Upon opening the application, you'll see the following features:

Toggle Light/Dark Mode: Click the light bulb icon to switch between light and dark themes.
Task Management:
-Create Task: Click the "+ Add Task" button to open a form and fill in the details.
-Mark as Completed: Click on the task to mark it as completed, which will update its status.
-Delete Task: Click the delete button next to any task to remove it.

Additional Features:
-Light/Dark Mode Toggle: Users can switch between light and dark modes for a personalized experience.
-Task Management: Create, delete, and mark tasks as completed with intuitive buttons and notifications.
-Responsive Design: The application is optimized for various screen sizes, ensuring usability on mobile and desktop devices.
-Custom Hooks: Utilizes useTaskService for efficient task-related API interactions, enhancing code reusability.
-User-Friendly Alerts: Integrated toast notifications provide feedback on user actions (e.g., task creation, deletion).

Technologies Used:
React: A JavaScript library for building user interfaces.
TypeScript: A typed superset of JavaScript that compiles to plain JavaScript.
Context: A predictable state container for JavaScript apps.
Axios: A promise-based HTTP client for making API requests.
React Icons: A library for including icons in the application.
SCSS: A preprocessor for CSS that allows for nested rules and variables.
Antd: An React Component to make UI friendly.
